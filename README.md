# Python Day 2 and Day 3 Notes

Simple Hinglish classroom notes for Python Day 2 and Day 3.

## Files

- `index.html` - Browser page with sidebar navigation for Day 2 and Day 3.
- `day2_python_fundamentals_notes.md` - Detailed Day 2 notes with examples, output, and explanation.
- `day3_operators_conditions_loops_notes.md` - Detailed Day 3 notes with operators, conditions, loops, and practicals.
- `examples/day2_marks_calculator.py` - Runnable Day 2 marks calculator practical.
- `examples/day2_bill_calculator.py` - Runnable Day 2 bill calculator practical.
- `examples/day2_age_calculator.py` - Runnable Day 2 age calculator practical.
- `examples/day3_grade_and_table.py` - Runnable Day 3 grade and multiplication table practical.
- `examples/day3_even_odd_checker.py` - Runnable Day 3 even/odd practical.
- `examples/day3_sum_and_pattern.py` - Runnable Day 3 sum and pattern practical.

## How to Use

Open `index.html` in a browser. Use the sidebar to switch between Day 2 and Day 3 notes.

Run practical files:

```bash
python examples/day2_marks_calculator.py
python examples/day2_bill_calculator.py
python examples/day2_age_calculator.py
python examples/day3_grade_and_table.py
python examples/day3_even_odd_checker.py
python examples/day3_sum_and_pattern.py
```
