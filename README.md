# Python Day 2 and Day 3 Notes

Fresh simple Hinglish classroom notes for Python Day 2 and Day 3.

## Files

- `index.html` - Browser page with sidebar navigation for Day 2 and Day 3.
- `day2_python_fundamentals_notes.md` - Fresh Day 2 notes exactly topic-wise: explanation, use, benefit, practical code, output, and detailed line explanation.
- `day3_operators_conditions_loops_notes.md` - Fresh Day 3 notes exactly topic-wise: explanation, use, benefit, practical code, output, and detailed line explanation.
- `examples/day2_basic_practice.py` - Runnable Day 2 basic practice file.
- `examples/day3_pattern_practice.py` - Runnable Day 3 pattern practice file.

## How to Use

Open `index.html` in a browser. Use the sidebar to switch between Day 2 and Day 3 notes.

Run practical files:

```bash
python examples/day2_basic_practice.py
python examples/day3_pattern_practice.py
```
